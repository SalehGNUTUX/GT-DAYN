## ✨ المزايا الرئيسية

### 💳 إدارة الديون
- تتبع الديون في الاتجاهين: **لي عند** و**عليّ** مع إحصاء فوري للصافي
- إضافة دفعات جزئية وجدولة دفعات مستقبلية بتاريخ استحقاق
- سجل تفصيلي لكل تعديل على المبلغ أو النوع مع تاريخه والفارق
- أرشفة الديون المكتملة مع إمكانية الاستعادة في أي وقت
- بحث شامل بالاسم والهاتف والمبلغ والتاريخ والوصف

### 👥 ملفات الأشخاص
- صفحة تعريفية لكل شخص مع إحصائيات: لي عنده / عليّ له / الصافي
- نسخ رقم الهاتف بنقرة واحدة
- مشاركة تقرير كامل بكل الديون مع شخص معين

### 📊 الميزانية الشهرية
- تحديد دخل شهري وفئات إنفاق مخصصة بأيقونات إيموجي
- شريط تقسيم مرئي لمعرفة توزيع المصاريف دفعة واحدة
- تنبيهات بمواعيد استحقاق الفواتير الشهرية
- اقتراحات ذكية لتسديد الديون من الفائض الشهري

### 🧮 آلة حاسبة مدمجة
- زر حاسبة في كل خانة مبلغ مالي
- دعم العمليات المتسلسلة دون الحاجة لضغط = كل مرة
- سجل العمليات السابقة مع إمكانية إعادة استخدام أي نتيجة
- حاسبة مستقلة في قائمة الإعدادات السريعة

### 🔒 الخصوصية والأمان
- **بيانات محلية بالكامل** — لا سحابة، لا خوادم خارجية
- قفل التطبيق بكلمة مرور
- تصدير واستيراد نسخ احتياطية كاملة (JSON + SQLite)

### 🎨 التصميم والتجربة
- وضع مظلم وفاتح تلقائي حسب إعدادات النظام
- دعم كامل للغة العربية مع خط **Ubuntu Arabic**
- تصميم متجاوب يعمل على الهاتف والحاسوب
- 27 عملة عربية وعالمية مدعومة

## 🚀 طريقة التشغيل

### 🐧 Linux — AppImage

```bash
# 1. بعد التحميل، اجعل الملف قابلاً للتنفيذ
chmod +x GT-DAYN-1.0.0-linux-x86_64.AppImage

# 2. شغّله
./GT-DAYN-1.0.0-linux-x86_64.AppImage
```

> **نصيحة:** استخدم [Gearlever](https://flathub.org/apps/it.mijorus.gearlever) (Flathub) لتثبيت AppImage في قائمة تطبيقات نظامك تلقائياً.

### 🤖 Android — APK

1. حمّل ملف `GT-DAYN-1.0.0.apk`
2. فعّل خيار **"تثبيت من مصادر غير معروفة"** في إعدادات هاتفك
3. افتح الملف وأكمل التثبيت

## 📦 محتوى الحزمة

الملف يضم كل شيء ويعمل بدون إنترنت بعد أول تشغيل:

- واجهة المستخدم الكاملة (HTML / CSS / JavaScript)
- خط **Ubuntu Arabic** لعرض النصوص العربية بشكل مثالي
- مكتبة **sql.js** لقاعدة بيانات SQLite محلية
- مكتبة **Chart.js** للرسوم البيانية
- جميع الأيقونات والموارد

## 🛠️ البناء من المصدر

```bash
# المتطلبات: Node.js 18+
git clone https://github.com/SalehGNUTUX/GT-DAYN.git
cd GT-DAYN
npm install

# تشغيل نسخة الويب (PWA)
npm run dev

# بناء AppImage
npm run electron:linux

# بناء APK
npm run android:apk
```

راجع [`docs/BUILD.md`](docs/BUILD.md) للتفاصيل الكاملة.
