كيف تربط حسابك بـ Google Drive؟
  1. افتح console.cloud.google.com
  2. أنشئ مشروعاً جديداً ← APIs & Services
  3. فعّل Google Drive API
  4. أنشئ OAuth 2.0 Client ID ← Web application
  5. أضف ${location.origin} في Authorized Origins
  6. انسخ الـ Client ID وضعه في DriveService.js
