# مجلد الخطوط — GT-DAYN

ضع ملف الخط في هذا المجلد بالاسم الدقيق:

```
fonts/Ubuntu Arabic Regular.otf
```

الخط المستخدم: **Ubuntu Arabic Regular**  
التنسيق: `.otf` (OpenType)

---

## لماذا الخط المحلي؟

التطبيق يعمل بشكل كامل **بدون إنترنت** بعد أول تحميل.  
الخط المحلي يضمن عرض النصوص العربية بشكل صحيح حتى في وضع عدم الاتصال.

## الخط الاحتياطي

إذا لم يُعثر على الخط المحلي، يستخدم التطبيق خط **Cairo** من Google Fonts  
(يُخزَّن تلقائياً بواسطة Service Worker عند أول تحميل بإنترنت).
